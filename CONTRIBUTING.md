# Contributing to ForeMetric

## Bounty Program

ForeMetric offers $FORE token rewards for 
verified contributions.

### Rules

**Human contributors only.**
Automated submissions generated by AI bounty 
agents are rejected without review.

**Payment in $FORE only.**
All bounty rewards are paid exclusively in 
$FORE tokens on TON blockchain at Fair Launch 
Q3 2026. USDT, TRC20, ETH or any other token 
payments are not offered.

**One submission per contributor per issue.**
Multiple comments claiming the same bounty 
will be treated as spam.

**Quality over speed.**
We review code quality, test coverage, and 
architectural alignment — not submission time.

### How to contribute

1. Comment on the issue with your approach
2. Wait for acknowledgment before starting work
3. Submit a PR with full test coverage
4. Await code review and merge decision
